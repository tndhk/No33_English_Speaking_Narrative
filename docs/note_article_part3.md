## 技術的なこだわり：品質を支える仕組み

### マルチモデル対応 + フォールバック

このアプリの核は、**生成AI**です。

ただし、単一のAIモデルに頼るのは危険です。なぜなら、特定のサービスが停止することもあれば、価格が値上がりすることもあるから。

だから、このアプリは、**複数のAIモデルを段階的に使う**設計になっています：

```
【第1選択】Google Gemini 2.5 Flash
 ├─ 特徴：高速・高品質・コスト効率的
 ├─ 処理時間：1-2秒
 └─ 成功率：約95%

【第2選択】DeepSeek V3 / R1
 ├─ 特徴：論理的思考に強い
 ├─ 処理時間：2-3秒
 └─ 成功率：約90%

【第3選択】xAI Grok Beta
 ├─ 特徴：ユーモアと正確性のバランス
 ├─ 処理時間：2-3秒
 └─ 成功率：約88%
```

**もし Gemini が使えなくなっても、自動的に DeepSeek に、それでも駄目なら Grok に切り替わります。**

つまり、ユーザーは「このAIサービス、今日は調子悪いな」という煩わしさを感じることなく、いつも同じクオリティのサービスが受けられます。

### 厳格な品質検証

AIが生成したものをそのまま使うわけではありません。**自動的に品質チェック**が走ります：

#### ① スキーマ検証

生成結果が以下の形式を満たしているか確認：
✅ 英文ナラティブ（必須）
✅ キーフレーズ（1-5個）
✅ 代替表現（0-2個）
✅ リコールテスト（必須）
✅ 発音ヒント（オプション）

#### ② 文数チェック

ユーザーが「Normal（5-8文）」を選んだなら、生成された英文が5-8文か確認。許容範囲は±1文。

#### ③ 日本語混入チェック

英文のはずなのに、日本語がたくさん混じっていないか？テキスト全体の10%以上の日本語は NG。

#### ④ 論理的一貫性チェック

ユーザーの入力に対して、生成された英文が論理的にズレていないか？

**このような厳格なチェックを通らないと、ユーザーに返されません。**品質が落ちたら、自動的に別のAIモデルで再度生成します。

### モダンな技術スタック

技術的には、最新のベストプラクティスを取り入れています：

#### フロントエンド

```
📱 Vite 6.0（ビルドツール）
   従来の webpack より3倍以上高速

🎨 Vanilla JavaScript（フレームワーク不使用）
   React・Vue など重いフレームワークを使わず、軽量・高速

🔤 Inter・Outfit フォント
   読みやすさと国際対応を重視
```

#### バックエンド

```
⚡ Cloudflare Pages Functions（サーバーレス）
   サーバー管理がゼロ、世界中のエッジで実行（低遅延）

🗄️ Supabase（PostgreSQL + Auth）
   セキュアなユーザー認証と行ベースのセキュリティ設定
```

#### AI統合

```
各モデルのAPI統合
 ├─ フォールバックロジック
 ├─ レート制限管理
 └─ コスト最適化
```

**なぜこんな構成にしたのか？**

- **Vite + Vanilla JS** → モバイル最適化。読み込み速度が速い
- **Cloudflare Functions** → グローバルな遅延削減
- **Supabase** → Firebase より安い・自由度高い・開発効率的
- **マルチモデル対応** → 単一障害点を排除

### プライバシー重視設計

**「あなたのデータは、あなたのもの」** その原則の下、設計されています：

#### Row Level Security（RLS）

データベースレベルで、「あなたのデータだけ見える」という制限を実装。万が一セキュリティ脆弱性があっても、他人のデータは見えません。

#### ローカル開発対応

開発者向けに、すべてのシステムをローカルで動かせるようにしています。ユーザーのデータが自分のマシンに完全に留まる環境で、開発・テストできます。

#### データエクスポート機能

いつでも、あなたのデータを JSON / CSV / Markdown 形式で取り出せます。

## 学習理論に基づいた効果データ

上記の設計思想は、単なる機能ではなく、すべて学習理論に基づいています。

### 文脈記憶による定着率の向上

心理学研究によると、**文脈を持つ情報の方が、文脈のない情報より、はるかに記憶に残りやすい**ことが知られています。

```
【研究例】
- 無関連な単語をランダムに覚える：1週間後の記憶定着率 20%
- 意味のあるストーリーの中に埋め込まれた単語：1週間後の記憶定着率 85%
```

このアプリは、あなたの「体験」という強い文脈の上に、表現を乗せます。

結果として、**定着率が 5倍近くに向上する可能性**があります。

### アウトプット効果による習得の加速

言語習得研究の第一人者・Merrill Swain は、以下の仮説を提唱しました：

> **「インプット（読む・聞く）だけでなく、アウトプット（話す・書く）があることで、初めて言語が習得される」**

このアプリは、「書く→思い出す→声に出す」という、3段階のアウトプット機会を提供します。

結果として、**話す力が、従来の「読み・聞き」中心の学習より、格段に伸びやすくなります。**

### 間隔反復による長期記憶の形成

Ebbinghaus の忘却曲線理論によると、**「忘れかけたタイミングでの復習」が、最も効果的**です。

```
学習直後：100% の知識
1日後：40% に低下
→ ここで復習 → 70% に回復

3日後：50% に低下
→ ここで復習 → 80% に回復

7日後：60% に低下
→ ここで復習 → 90% に回復

30日後：70% に低下
→ ここで復習 → 95% に定着
```

このアプリの SRS は、この理論に基づいて、最適なタイミングで復習を促します。

結果として、**長期的な記憶定着が、飛躍的に向上します。**

### 3ヶ月使った場合の想定効果

```
【入力データ】
- 毎日 1 エントリー
- 3 ヶ月 = 90 エントリー
- 各エントリーから平均 4 個の重要表現を学習
- 合計 360 個の表現を学習

【復習データ】
- SRS に基づいて、毎日 5-10 個の表現を復習
- 3 ヶ月間で、合計 900-1200 回の復習セッション

【期待される効果】
✅ 学習した表現が、「知識」から「スキル」に変化
✅ 実際の会話で、無意識に表現が出てくるようになる
✅ 英語での「話すハードル」が、著しく低下
✅ 英語日記を書く習慣が、確実に定着
✅ 自分の成長が、カレンダーに可視化される
```

これは、決して大げさな予測ではなく、学習理論に基づいた、実現性の高い見通しです。

## 開発者からのメッセージ

### なぜ、このアプリを作ったのか

このアプリを作ったのは、私自身が「話せない中級者」だったから。

TOEIC 800 点を取得した。それなのに、英語を話せない──。

参考書で学んだフレーズを覚えても、実際の会話では「あれ、なんて言うんだっけ？」と口ごもってしまう。添削されない英作文には自信が持てない。続かない。

そんな悔しさと、「これを解く方法があるはずだ」という信念の下、このアプリを開発しました。

### ユーザーの声

今、このアプリを毎日使っているユーザーたちから、こんなメッセージをもらいます：

> 「フレーズ暗記より、ずつと定着しています」

> 「毎日 5 分だから、続けられます」

これらのメッセージが、開発を続ける最大の動機です。

### このアプリの本質

**このアプリは、単なる商品ではなく、多くの「話せない中級者」の悩みを解くために作られた、ある種の「社会的課題解決ツール」です。**

あなたが、そのツールを使ってくださることで、あなたの英語が変わるだけでなく、このアプリもまた進化します。

## 今、始めるべき理由

### 理由1：習慣形成には時間が必要

毎日 1 つのエントリーを書く習慣をつけるには、最低でも 3 週間は必要です。

3 ヶ月継続すれば、確実に変化を感じられる。

でも、その 3 ヶ月を「後で」に先延ばしすると、いつまでも変わりません。

### 理由2：言語習得に「近道」はない

残念なことに、英語を話す力を身につけるのに、魔法のような方法はありません。

ただ、「効率的な方法」は存在します。

このアプリは、その「効率的な方法」の集大成です。

でも、いくら効率的でも、**使わなければ効果はゼロ**です。

### 理由3：あなたの「今」は、二度と戻らない

「去年の今頃、始めていれば…」という後悔は、多くの人が持っています。

でも、そういう後悔は、**「今、始める」ことで、未来に作らずに済みます。**

1 年後、あなたは確実に成長しています。

その成長を手に入れるかどうか、決めるのは、**あなた次第**です。

## さあ、始めましょう

長い3回シリーズをお読みいただき、ありがとうございました。

ここまで読んでくださったあなたは、すでに「このアプリを使ってみたい」という気持ちが、心のどこかにあるはずです。

その気持ちを、**「行動」に変えるなら、今です。**

### 最初の一歩は、シンプル

**ステップ1：ログイン**
アプリにアクセスして、メールアドレスで登録します。（招待制ではなく、誰でも登録可能です）

**ステップ2：カテゴリーを選ぶ**
「今日の出来事」「考えたこと」「自由テーマ」から選びます。

**ステップ3：日本語で書く**
ガイド質問に答えて、日本語で 2-3 行書きます。

**ステップ4：生成を待つ**
「生成」ボタンを押して、5-10 秒待ちます。

**ステップ5：英文を確認する**
AI が自動生成した英文、フレーズ、リコールテストが表示されます。

**これだけです。所要時間：約 5 分。**

### 最後に

「英語が話せたら…」

多くの人が、そんな想像をしたことがあると思います。

- 海外出張で、自由に意見が言える
- 映画を字幕なしで理解できる
- 外国人の友人と、深い会話ができる
- キャリアの選択肢が広がる

その「たら」が、「ら」を取って、「英語が話せる」に変わる瞬間は、本当に近いかもしれません。

**あなたが、今、この一歩を踏み出すなら。**

さあ、始めましょう。
