# 日本語2行で英語が学べる？『英語日記アプリ』の使い方を実例で解説

## リード文

前回の記事では、「TOEIC 800でも話せない」という課題と、その解決策としての**設計思想**をお伝えしました。

今回は、いよいよ**具体的な使い方**を実例で紹介します。

「日本語2行」「あとはAIにおまかせ」という謳い文句は、本当なのか？実際の流れを、朝の10分から始まる1つのストーリーで見ていきましょう。

---

## 使い方ガイド：1日の学習ストーリー

### ステップ1：カテゴリーを選ぶ

**朝、仕事に行く前。あなたはアプリを開きます。**

まず最初に見えるのが、3つのカテゴリー選択画面です：

- 📝 **今日の出来事** - 日中に起きたことを記録
- 💭 **考えたこと・思ったこと** - 自分の考えや意見を表現
- ✨ **自由テーマ** - 何でもOK、好きなことを書く

「今日の出来事」を選んだとします。

### ステップ2：ガイド質問に答える（日本語）

すると、ガイド質問が出てきます：

> **「昨日、何があった？」**
>
> あなたの回答：「プレゼンテーション失敗。資料の誤字に気づかず、上司に指摘された。落ち込んだ。」

次の質問：

> **「どう感じた？」**
>
> あなたの回答：「ショックだったけど、もっと注意深くなる機会だと思う」

最後の質問：

> **「今、どう思ってる？」**
>
> あなたの回答：「次はしっかり確認してから提出しよう」

**これで入力完了です。日本語で、合計3-4行。**所要時間：約2分。

### ステップ3：生成ボタンを押す

「生成」ボタンをタップすると、5-10秒待ちます。

### ステップ4：5つの学習素材が自動生成される

AI がお手上。 を読んで、自動的に以下の5つが生成されます：

#### ① 英文ナラティブ（あなたの日記）

```
Yesterday, I gave a presentation at work, but I made a careless mistake.
I didn't notice a typo in my slide until my supervisor pointed it out.
I felt shocked at first, but I'm starting to see it as an opportunity to be more careful.
Moving forward, I'll double-check everything before submitting.
```

自然で、あなたの文脈に沿った英文です。

#### ② キーフレーズ（5個まで）

```
🔹 "give a presentation"
   意味：プレゼンテーションをする
   使い方：会議や説明の場面で。単に "present" より自然

🔹 "careless mistake"
   意味：うっかりした誤り
   使い方：自分の責任による小さなミスに

🔹 "point it out"
   意味：それを指摘する
   使い方：誰かが問題を見つけて教える時に

🔹 "see it as an opportunity"
   意味：それを機会として見る
   使い方：ネガティブなことをポジティブに捉える

🔹 "double-check"
   意味：二重確認する
   使い方：ミスを防ぐために、もう一度確認する
```

単に意味だけでなく、「どういう文脈で使うのか」という情報も含まれています。

#### ③ 代替表現（2個まで）

```
💡 代替1: "I failed to notice..." instead of "I didn't notice"
   ニュアンス：より正式・文語的な言い方

💡 代替2: "My supervisor brought it to my attention"
   ニュアンス：さらに丁寧な表現。ビジネスレター向け
```

同じ意味でも、シーンによって使い分けられる表現を提供します。

#### ④ リコールテスト（復習問題）

```
【日本語プロンプト】
「昨日のプレゼンで、資料の誤字に気づかなかった。
それをあなたの上司が指摘した。
このシーンを、英語で説明してみてください」

【解答例】
"Yesterday, I gave a presentation at work, but I made a careless mistake.
I didn't notice a typo in my slide until my supervisor pointed it out."
```

後日の復習時に、このテストが出題され、あなたが「英語で再現できるか」をチェックします。

#### ⑤ 発音ヒント

```
📢 "presentation" [ˌprezənˈteɪʃən]
   アクセント：プレゼンTEIション（第2音節）

📢 "careless" [ˈkeərləs]
   アクセント：CARE・less（第1音節）
```

難しい単語には、IPA記号とアクセント位置を記載。

### ステップ5：音声で聴く（シャドーイング練習）

生成された英文を見た後、**クリック1つで聴くことができます。**

- **文単位で再生可能** - 1文目だけ、2文目だけ、という単位で再生
- **速度調整可能** - 0.9倍速、通常速、1.1倍速から選べる
- **リピート再生** - 同じ文を何度も繰り返す

朝の移動中に聞きながら、英文を「追いながら言ってみる」（シャドーイング）ことができます。

**所要時間：約3分。**

## 次のステップ：復習（1日後から自動で始まる）

### 翌日：復習通知が来る

あなたのスマホに通知が来ます：

> 「『今日の出来事』の復習：今日は1件のエントリーがあります」

アプリを開くと、復習タブに昨日のエントリーがリストアップされています。

### リコールテストに挑戦

リコールテスト画面が表示されます：

```
【日本語プロンプト】
「昨日のプレゼンで、資料の誤字に気づかなかった。
それをあなたの上司が指摘した。
このシーンを、英語で説明してみてください」
```

あなたが頭の中で、または口に出して英語を作ってみます。

その後、「答えを見る」をタップすると：

```
【解答表示】

"Yesterday, I gave a presentation at work, but I made a careless mistake.
I didn't notice a typo in my slide until my supervisor pointed it out."
```

### 自己評価で復習スケジュールが決まる

重要なのが、**自己評価画面**です：

```
🟢 覚えていた（Easy）
🟡 かなり頑張って思い出した（Good）
🔴 うろ覚えだった（Hard）
⚫ 全く覚えていなかった（Forgot）
```

- 「覚えていた」→ 次の復習は7日後
- 「頑張って思い出した」→ 次の復習は3日後
- 「うろ覚え」→ 次の復習は1日後
- 「忘れた」→ 次の復習は翌日

**つまり、あなたの理解度に応じて、自動的に最適な復習スケジュールが決まります。**

## 他の便利機能

### 「年前の今日」機能で成長を実感

カレンダーを開くと、過去の同じ日付のエントリーが見える。例えば：

```
【今年の12月27日】プレゼンで失敗した話
【1年前の12月27日】初めてのプレゼンに緊張していた話
```

1年前は「何も話せず、どう言ったらいいのかわからない」という悩みを抱えていたのに、今年は「失敗したけど、工夫で次に活かそう」という成長が見えます。

**この「成長の実感」が、最高のモチベーション**になります。

### 統計情報でモチベーション維持

ダッシュボードには、こんな情報も表示されます：

```
✍️ 合計エントリー：127件
🔄 復習待ちの今日：5件
⭐ 習熟済み（マスター）：34件
📅 連続ストリーク：23日
```

「23日連続」を途切れさせたくないから、「今日も書こう」という気持ちになる。

## 実際に使ってみた3つの変化

### ① 思い出す経験が、劇的に増える

従来の Anki では、単語カードを見ただけの単純な反応。

でも、このアプリでの復習は違います。**「昨日のプレゼンでの失敗」という文脈が呼び起こされ、その時の感情、状況が蘇ります。**その中で表現を再構築する。

結果として、記憶が格段に強くなる。

### ② 実際の会話で、無意識に出てくる表現が増える

これが、最大の効果です。

アプリで学んだ表現が、「思い出す」を強要されることなく、**自然に口から出る**という現象が起きます。

### ③ 書く習慣が、無理なく身につく

最初は「毎日これをやるのか」と思っていても、1週間もすると習慣化します。

理由は：

- **ハードルが低い** - 日本語で2-3行書くだけ
- **毎日1つだけ** - 迷わない
- **即座に完成** - 書いたら、すぐに英文と学習素材ができている

その結果として、気づいたら「あ、もう50日連続書いてる」という状態になっています。

## 所要時間まとめ

```
初回エントリー作成：5分
  ├─ ガイド質問に答える：2分
  ├─ 生成を待つ：1分
  └─ 英文と素材を確認：2分

毎日の復習：3-5分
  ├─ リコールテストに答える：2分
  ├─ 答えを確認して評価：1分
  └─ TTS で音声を聴く：1-2分

3ヶ月継続した場合：
  毎日5分 × 90日 = 約450分（7.5時間）
  その見返り：360個の表現を習得＆習慣化
```
